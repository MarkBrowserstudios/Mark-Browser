# Mark Browser

Navegador web moderno com design Mica, construído com Electron.

## Design

- **Tema Mica**: Interface semi-transparente com blur e efeitos de vidro
- **Tema Verde Escuro**: Paleta de cores verde neon sobre fundo escuro transparente
- **Barra Lateral**: Acesso rápido a apps (Telegram, Skype, Instagram, Gmail)
- **Logo Tecnológica**: SVG animada com gradiente verde e efeito glow

## Funcionalidades

- Interface Mica com transparência e blur
- Barra lateral com apps predefinidos + botão para adicionar sites
- Múltiplas abas com gerenciamento completo
- Barra de endereços inteligente
- Atalhos rápidos na página inicial
- Favoritos com persistência local
- Controles de janela personalizados
- Atalhos de teclado

## Estrutura


## Atalhos

| Atalho | Ação |
|--------|------|
| Ctrl+T | Nova aba |
| Ctrl+W | Fechar aba |
| Ctrl+L | Focar barra de endereço |
| F5 | Recarregar |
| Alt+← | Voltar |
| Alt+→ | Avançar |

## Apps na Barra Lateral

- Telegram (web.telegram.org)
- Skype (web.skype.com)
- Instagram (instagram.com)
- Gmail (mail.google.com)

Clique em **+ Add Site** para adicionar mais apps.
